Cartify API Documentation

Welcome to the ShopKart API documentation. This API allows you to interact with ShopKart programmatically to manage products, users, orders, and more.

API Overview

The Cartify API is a RESTful API that uses JSON for requests and responses. All API access is over HTTPS, and the base URL for all requests is:

https://api.shopkart.com/v2

The API uses standard HTTP response codes to indicate success or failure of requests. Authentication is required for most endpoints, which can be done using API keys.

Base URL

All URLs referenced in the documentation have the following base:

https://api.shopkart.com/v2

Response Format

All API responses are returned in JSON format, including errors. Successful responses will have a structure similar to:

{
  "status": "success",
  "data": {
    // Response data here
  },
  "message": "Operation completed successfully"
}

Authentication

GET

Most ShopKart API endpoints require authentication using API keys. You can generate API keys from your ShopKart dashboard.

Using API Keys

Include your API key in the request header as follows:

Authorization: Bearer YOUR_API_KEY

Example Request

cURL
JavaScript
Python
curl -X GET \
  -H "Authorization: Bearer sk_test_123456789" \
  "https://api.shopkart.com/v2/products"
const response = await fetch('https://api.shopkart.com/v2/products', {

headers = {
  'Authorization': 'Bearer sk_test_123456789'
}

response = requests.get('https://api.shopkart.com/v2/products',

List Products NEW

GET

Retrieve a list of products from your ShopKart store. The products are returned sorted by creation date, with the most recent products appearing first.

Endpoint

GET /products

Parameters

Parameter Type Description
limit integer Number of products to return (default: 20, max: 100)
offset integer Number of products to skip (for pagination)
category string Filter products by category ID
in_stock boolean Filter products by availability

Example Request

curl -X GET \
  -H "Authorization: Bearer sk_test_123456789" \
  "https://api.shopkart.com/v2/products?limit=10&in_stock=true"

Example Response

{
  "status": "success",
  "data": {
    "products": [
      {
        "id": "prod_123",
        "name": "Premium Headphones",
        "price": 199.99,
        "in_stock: true
      },
      // ... more products
    ],
    "total": 150,
    "has_more": true
  }
}

Create Product UPDATED

POST

Create a new product in your ShopKart store. All required fields must be provided in the request body.

Endpoint

POST /products

Request Body

Parameter Type Description
name string Required. The product's name
price number Required. The product's price
description string The product's description
category_id string ID of the product's category
in_stock boolean Whether the product is in stock (default: true)

Example Request

curl -X POST \
  -H "Authorization: Bearer sk_test_123456789" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "New Product",
    "price": 49.99,
    "description": "A great new product",
    "category_id": "cat_123",
    "in_stock": true
  }' \
  "https://api.shopkart.com/v2/products"

Example Response

{
  "status": "success",
  "data": {
    "product": {
      "id": "prod_456",
      "name": "New Product",
      "price": 49.99,
      "description": "A great new product",
      "category_id": "cat_123",
      "in_stock": true,
      "created_at": "2023-07-15T10:30:00Z"
    }
  },
  "message": "Product created successfully"
}
